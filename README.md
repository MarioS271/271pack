# 271pack
## This minecraft resource pack was created by me using [vanillatweaks](https://vanillatweaks.net/picker/resource-packs/) and some self-made resources.

Features:
  - Inbuild resource efficient shaders
  - New Button & GUI design (Dark Theme)
  - Title screen & menu themes
  - High-Resolution Font
  - Bug fixes like corrected item holding
  and much more!

### NOTE: THE RESOURCE PACK NEEDS FANCY GRAPHICS FOR SOME THINGS TO WORK PROPERLY!!!
#### UPDATE TO VERSION 6.0: Replaced High Contrast Theme with Dark Theme, added variated textures for menu background depending on current theme
