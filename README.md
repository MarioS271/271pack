# 271pack
### This minecraft resource pack was created by me using [vanillatweaks](https://vanillatweaks.net/picker/resource-packs/) and [HighContrastExtended](https://www.planetminecraft.com/texture-pack/high-contrast-extended/).

Features:
  - 3D Blocks, Items and more
  - Some fixes like corrected item holding
  ... and much more!


Variants of the resourcepack with different menu backgrounds are provided.

Background images from:
  - [WallpaperAccess](https://wallpaperaccess.com/minecraft)
  - [Google](https://www.google.com/search?q=hd+minecraft+wallpapers)
