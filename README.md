# 271pack
## This minecraft resource pack was created by me using [vanillatweaks](https://vanillatweaks.net/picker/resource-packs/) (f.E.: 3D Item Models, Dark UI, ...) and a few self-made resources (mostly HUD stuff).
<br>
Features:
  - Inbuild resource efficient shaders (Wavy & Bushy Plants, Wavy Water)
  - New Button & GUI design (Dark Theme)
  - Title screen & menu themes
  - High-Resolution Font
  - Bug fixes like corrected item holding
  and much more!
<br>
<br>
## IMPORTANT: THE RESOURCE PACK NEEDS FANCY GRAPHICS FOR SOME THINGS TO WORK PROPERLY!!!
